# 🐍 可愛貪食蛇 (Cute Snake Game) ✨

這是一個專為女孩子設計的、充滿粉紅泡泡風格的貪食蛇遊戲！採用 HTML5 Canvas、CSS3 與原生 JavaScript 開發。

## 🎨 遊戲特色
- **繽紛美學**：精選櫻花粉與薰衣草紫配色，搭配流暢的毛玻璃效果。
- **可愛元素**：蛇蛇有腮紅跟眼睛，食物是各種美味的甜點（蛋糕、草莓、甜甜圈）。
- **多段速度**：提供「慢慢」、「正常」、「衝刺」三種難度，適合不同強度的反應挑戰。
- **響應式設計**：支援電腦方向鍵與手機觸控操作。

## 🎮 如何遊玩
1. 開啟遊戲網址。
2. 選擇你喜歡的速度。
3. 點擊「開始玩 💖」。
4. 使用方向鍵控制蛇蛇去吃甜點，獲得最高分！
