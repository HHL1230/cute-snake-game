# 🐍 可愛貪食蛇 (Cute Snake Game) ✨

這是一個超級可愛的貪食蛇遊戲！採用 HTML5 Canvas、CSS3 與原生 JavaScript 開發。

## 🎨 遊戲特色
- **動態配色主題**：淡藍、淡黃、粉紅、粉綠四種風格隨機切換
- **多種動物頭像**：🐱🐶🐰🐭🐻🐼🦊🐹🐸 每次遊戲隨機出現
- **繽紛身體顏色**：6 種粉嫩顏色隨機搭配
- **高解析度移動**：600x600 畫布 + 40x40 細緻格子
- **多段速度**：「慢慢」、「正常」、「衝刺」三種難度
- **甜點食物**：🍓🍰🍪🍩🧁🍭🍬💖⭐🌸

## 🎮 如何遊玩
1. 開啟遊戲網址或 `index.html`
2. 選擇你喜歡的速度
3. 點擊「開始玩 💖」
4. 使用方向鍵控制小動物去吃甜點，獲得最高分！
